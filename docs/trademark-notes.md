# Brand Asset Notes

These notes are intentionally secondary to the public brand showcase. Use them only when a project needs a simple reminder about licensing or brand-asset handling.

## Simple Usage

Use **McGuire Technology™** in public-facing brand references, especially in:

- website pages
- repository READMEs
- proposals and presentations
- client-facing documentation
- product announcements
- official templates

Use the plain name **McGuire Technology** in repeated body copy after the first prominent reference when the trademark symbol would make the writing feel cluttered.

## Symbol Guidance

Use:

- **McGuire Technology™**

Do not use:

- **McGuire Technology®**

The federal registration symbol should only be used after registration has actually been issued by the USPTO.

## License Declaration

The McGuire Technology name, logo, marks, icons, visual identity, and official brand materials are proprietary unless explicitly released under a separate written license.

Reusable technical assets can be licensed differently from brand assets. Keep that distinction clear in repositories and public documentation.

## Covered Brand Assets

The McGuire Technology name, logo, wordmarks, icons, product marks, visual identity, and official brand materials are protected brand assets of McGuire Technology, LLC.

## Technical Materials

The main repository license applies only to non-brand code, templates, configurations, scripts, and other reusable technical materials.
